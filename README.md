# weather-app
with react-native expo 


### expo
```
npm i -g expo-cli@3.13.1
```

### expo modules
```
expo install expo-location
expo install expo-permissions
expo install expo-linear-gradient
```

### axios
```
npm install --save axios
```

### prop-types
```
npm i prop-types
```
